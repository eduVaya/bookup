# BookUp — Frontend

React + TypeScript frontend for BookUp, a full stack book club management application.

🔗 **Live:** [bookup-jade.vercel.app](https://bookup-jade.vercel.app)

## Tech Stack

- **Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Data Fetching:** TanStack Query
- **Routing:** React Router v7
- **File Upload:** Cloudinary
